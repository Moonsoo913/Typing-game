# Typing-game
Typing practice game(Clone Coding)

2초안에 화면에 표시되는 단어를 입력하면, 한 단어당 5점씩 얻을 수 있는 타자 연습 게임입니다.

## Usage
```
> git clone https://github.com/moonchop/Typing-game.git
> cd Typing-game
> Open with live server(index.html)
```
